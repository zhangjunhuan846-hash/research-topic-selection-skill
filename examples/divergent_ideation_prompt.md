# Divergent Ideation Prompt Example

Use this prompt when generated topics are too similar or overly concentrated on one variable path.

Prompt:

Please enter Divergent Ideation Mode.

My broad field is [material/system/application]. Existing topic generation has become too concentrated around [crowded axis A], [crowded axis B], and [crowded axis C].

Known crowded axes:
- [axis already heavily studied]
- [axis already heavily studied]
- [generic optimization without a falsifiable mechanism]
- [broad screening without controlled comparison]

Please brainstorm from 8 independent lenses:
1. precursor or input chemistry
2. processing pathway
3. structure and accessibility
4. surface/interface mechanism
5. operating protocol or environment
6. engineering relevance
7. descriptor/statistical methodology
8. cross-system transferability

For each lens, generate 5 candidate directions.

For each candidate, include:
- English working title
- Chinese explanation
- core scientific question
- primary mechanism axis
- why it differs from the crowded axes
- collision risk
- feasibility score
- novelty score
- advisor-discussion value
- pilot suitability
- minimum validation needed

Do not directly select a final topic.
First generate a topic pool, then perform a light collision pre-screen, then select 3–5 candidates for deeper evaluation.
